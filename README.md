## SecureUxTheme
- Install SecureUxTheme
- Export Niivu's Theme in 'C:\Windows\Resources\Themes'
- Select your preferred Light Variant

## Nilesoft Shell
Use Full config given here as some symbols are changed due to conflicts with the explorer font changer mod!

## Windhawk
### Mods needed:
- Remove File Explorer Suffixes
- Taskbar Clock Customization
- Taskbar height and icon size
- Taskbar Labels for Windows 11
- Taskbar Thumbnail Size
- Windows 11 Notification Center Styler
- Windows 11 Start Menu Styler
- Windows 11 Taskbar Styler
## Explorer Patcher
### Taskbar:
Keep original W11 Taskbar
### In File Explorer section:
- turn on "Always use legacy file transfer dialog"
- "Disable modern search bar"
- Pick "Control Interface * : Windows 7 Command Bar" and "Title bar : Hide icon and title"
